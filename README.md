# DealHunt Frontend

> Your one-stop destination for the best deals — web prototype + Flutter app.

---

## Project Structure

```
DealHunt_Frontend/
│
├── web-prototype/          # HTML/CSS/JS prototype
│   ├── html/               # Page markup (entry: index.html)
│   ├── css/                # Stylesheets
│   ├── js/                 # JavaScript logic
│   ├── assets/
│   │   ├── images/         # Raster images
│   │   └── icons/          # Icon assets (SVG / PNG)
│   └── pages/              # Additional HTML pages
│
├── flutter-app/            # Flutter mobile/web app
│
├── docs/                   # Design specs, API docs, notes
│
├── .gitignore
└── README.md
```

---

## Getting Started

### Web Prototype
Open `web-prototype/html/index.html` in any modern browser — no build step required.

### Flutter App
```bash
cd flutter-app
flutter pub get
flutter run
```
